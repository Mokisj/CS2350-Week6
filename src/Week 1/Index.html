CS 2350: Client Side Web Development

HTML5

HTML 5 is a language used to structure content on the web. A few supporting examples are: multipmedia, advanced gragphics, native forms and semantic elements, all of which are used for improving accessiblilty and optimization. It provides a versitile platform for building web applications and content.

CSS3

CSS3 is what is a styling language for describing the presentation of HTML and other documents on the web. It is used for advanced layout techniques providing a variety of layouts to customize a website. CSS3 can also be used to create multiple different layouts for different sized screen depending on how big the users screen is.

JavaScript

JavaScript is a language for providing interactive and dynamic web content. Executed by web browsers it allows the dev to add dynamic functionality to web pages. It can also be used to create web based games and mobile applications. JavaScript is usually coupled with HTML and CSS to create websites.
